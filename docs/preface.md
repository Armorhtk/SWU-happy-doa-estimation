# 前言

这不是一份只堆术语的雷达课程站，而是一门尽量让本科生真正学会动手的 DOA 入门课。

## 这门课怎么学

- 先建立阵列、角度与空间响应的直觉
- 再进入经典超分辨率方法
- 最后再进入基于深度学习的 DOA 估计

## 你不需要一开始就会什么

你不需要先读完一堆论文，也不需要一开始就会 MUSIC 或深度学习模型。  
你只需要先跑通第一份实验，然后带着现象去理解概念。

## 这门课的目标

- 让本科生进入雷达 DOA 估计方向
- 让学习过程尽量代码驱动、实验驱动
- 为后续深入传统方法与深度学习方法打基础

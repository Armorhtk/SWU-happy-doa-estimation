# 关于项目

`SWU-happy-doa-estimation` 是一个面向本科生的开源课程项目，目标不是堆砌术语，而是把“能学懂、能跑通、能继续扩展”放在第一位。

## 项目原则

- 教学优先于炫技
- 文档优先于碎片化笔记
- 低成本维护优先于复杂平台能力
- 每个章节都要让学生实际动手

## 为什么选择 Docusaurus + GitHub Pages

- 免费托管
- 静态站点，长期维护成本低
- 文档组织能力强
- 后续增加章节时基本不需要改基础设施

## 当前状态

当前公开的是 `V1` 骨架版：

- 已完成课程路线与快速开始
- 已完成 DOA 入门样章
- 已提供第一份仿真实验代码
- 传统方法与深度学习部分仍在持续建设

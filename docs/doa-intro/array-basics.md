# 阵列、角度与波长

## 什么是 DOA

DOA 是 `Direction of Arrival`，也就是“信号从哪个方向来”。

如果你只有一个传感器，通常只能知道“有信号”；如果你有一排传感器，就能利用不同阵元接收到的相位差来推断方向。

## 为什么从均匀线阵开始

均匀线阵 `ULA` 的好处是结构简单：

- 每个阵元沿一条直线排开
- 相邻阵元间距固定
- 很多经典 DOA 方法都从这里起步

## 为什么常用半波长间距

如果阵元间距太大，扫描图里容易出现额外峰值，也就是常说的栅瓣。  
入门阶段先用 `lambda/2`，能减少很多干扰，让你把注意力放在主现象上。

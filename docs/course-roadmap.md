# 课程路线

这门课不是把你一次性塞进论文复现，而是先给你一个能跑通、能看懂、能改参数的学习闭环。

## 适合谁

- 西南大学或类似背景的本科生
- Python 初学者
- 想从雷达阵列信号处理入门，但不希望一开始被公式和工程环境劝退的同学

## 你在 V1 学完后能做到什么

- 搭好本地 Python 环境并运行课程代码
- 理解均匀线阵、波达方向、阵元间距和波长之间的关系
- 运行并修改第一组空间响应仿真实验
- 为后续进入 MUSIC、ESPRIT 和深度学习 DOA 做好代码基础

## 10 周建设路线图

### 第 1-3 周：DOA 入门

- 环境配置与课程使用方式
- 复数、向量、矩阵的最小预备知识
- ULA 几何直觉与扫描响应
- 第一个可运行仿真实验

### 第 4-7 周：传统超分辨率方法

- 协方差矩阵与子空间思想
- Bartlett 与 MUSIC 的对比
- 分辨率、快拍数、信噪比的影响
- 简化版实验复现与参数分析

### 第 8-10 周：基于深度学习的 DOA 估计

- 数据集构造思路
- PyTorch 训练流程
- 传统方法与学习方法对比
- 最小深度学习基线

## 学习方式建议

每个教学单元都按同一节奏组织：

1. 先看学习目标与先修要求
2. 跑通一份最小脚本
3. 观察图像和输出
4. 再回头理解关键概念
5. 完成练习并对照参考答案

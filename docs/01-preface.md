# 教程导览

`SWU-happy-doa-estimation` 是一个面向本科生的雷达 DOA 估计教程站。

这个项目的目标，不是一次性覆盖阵列信号处理的全部分支，而是提供一条清楚、可执行、能逐步建立结果感的学习主线。当前公开版本优先把第一章写实，先帮助读者建立阵列信号模型、常规波束形成空间谱估计以及最基本的结果解释能力。在此基础上，再逐步扩展到经典超分辨算法、深度学习 DOA 估计以及面向论文复现和毫米波雷达应用的专题内容。

## 适合谁阅读

本教程站主要面向以下读者：

- 电子、通信、雷达、信号处理等相关专业的本科生
- 已具备线性代数、概率论和数字信号处理基础，希望进入阵列信号处理方向的学习者
- 想用 Python 完成 DOA 估计实验、方法比较和初步研究训练的读者

## 当前版本你将获得什么

完成当前主线后，读者应当能够：

- 清楚 DOA 估计问题是什么，雷达阵列为什么能够感知信号来向
- 理解窄带远场阵列信号模型、均匀直线阵几何、导向矢量、快拍与协方差矩阵等基础概念
- 独立完成常规波束形成空间谱估计实验，并正确解释空间谱峰值的含义

## 教程特点

- 以主线组织内容：先把核心链条讲清，再逐步展开专题
- 以实验驱动理解：每一章都尽量落到可执行的 Python 实验
- 以比较建立判断：后续内容会强调统一评估标准和比较框架
- 保留生长空间：当前版本先明确主线，专题内容以后续更新方式逐步加入

## 教程内容导览

当前版本的内容安排，可以理解为三层：

### 已完成

- 第一章：阵列信号模型与常规波束形成

### 即将发布

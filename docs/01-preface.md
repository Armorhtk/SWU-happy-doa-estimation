# 导读

`SWU-happy-doa-estimation` 是一个面向本科生的 DOA 估计教程站。当前版本先把课程结构与学习路径对齐，再按章节逐步补全正文、实验和结果解释。

## 0.1 DOA估计的应用背景与核心问题

DOA（Direction of Arrival，波达方向）估计要回答的核心问题很直接：**阵列接收机怎样根据多通道观测，判断信号是从哪个方向来的。**

这个问题会出现在很多典型场景里：

- 雷达要根据阵列回波判断目标方位。
- 麦克风阵列要估计声源方向并支持波束形成。
- 无线通信系统要根据到达方向做定位、跟踪或空间滤波。

对初学者来说，真正需要先看清的不是“某个算法公式”，而是三件事：

- 阵列为什么比单通道观测多出方向信息。
- 数学模型怎样把路径差、相位差和协方差结构组织起来。
- 算法输出的角度、谱峰和误差指标应该怎样解释。

## 0.2 课程目标与学习路径说明

这门教程的目标不是一次性覆盖全部阵列信号处理内容，而是给本科生提供一条清楚、可执行、便于继续深入的入门主线。

读完当前大纲后，你应该逐步建立下面这条学习路径：

- 第一章先搭起 DOA 入门框架，理解阵列观测模型、远场窄带假设、协方差矩阵、空间谱以及分辨率极限。
- 第二章进入经典 DOA 估计算法，建立子空间方法、模型阶数选择、MUSIC、ESPRIT 与空间平滑的比较框架。
- 第三章再讨论深度学习 DOA 估计，重点放在任务定义、数据表示、分类与回归方法以及和经典方法的对照。
- 第四章面向论文复现和工程实践，把方法落到 Trans 论文、FMCW 毫米波雷达处理流程和实测验证。
- 附录补足数学基础、调试问题、资料入口，帮助你在阅读正文时及时回查。

如果你是第一次接触这个方向，建议按“导读 -> 第一章 -> 第二章 -> 第三章 -> 第四章 -> 附录”的顺序往下读；如果你已经具备一定基础，也可以直接从对应章节进入，再用附录做查漏补缺。

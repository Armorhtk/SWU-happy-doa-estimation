# FAQ

## 为什么 V1 不直接讲 MUSIC

因为对本科初学者来说，真正的难点通常不是记住算法名字，而是没有建立阵列和空间响应的直觉。  
如果这层直觉缺失，MUSIC 很容易学成“会调库，不会解释”。

## 为什么现在不做在线运行环境

因为这个项目强调零成本长期维护。  
把站点、环境、实验、部署都压在免费可控方案上，能让项目更容易持续更新。

## 为什么先用脚本，不主打 Notebook

脚本更适合版本管理，也更利于后续把实验逐步整理成可复用模块。  
后续如果需要交互式教学，再补充 Notebook。

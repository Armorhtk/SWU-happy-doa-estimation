# 3.8 深度学习方法与经典方法的对比与选型

本节正在开发中，后续内容将按教程架构逐步补充。

## 这一节会回答什么

- 深度学习方法和经典方法应该从哪些维度做公平比较。
- 在数据、算力、可解释性和部署条件不同的情况下，应怎样选型。
- 哪些场景更适合把深度学习方法作为增强，而不是完全替代经典方法。

# SWU-happy-doa-estimation

> 面向本科生的中文 DOA 估计教程站：从阵列信号处理基础到经典算法，再到深度学习与工程实践。

## 适合谁与学完你能得到什么

这个项目主要面向：

- 电子信息、通信工程、人工智能等方向的本科生
- 刚接触阵列信号处理与雷达DOA估计的同学

完成主要章节后，你会得到：

- 一套可复用的 DOA 基础认知框架（模型、假设、指标）
- 一条经典方法到深度学习方法的对比与选型思路
- 面向实验复现与工程落地的基本实践路径

## 教程内容概览

### 导读

- 课程定位、学习建议与整体路径说明

### 第一章 DOA估计入门（已发布）

- 建立阵列观测模型与导向矢量直觉
- 理解协方差、常规波束形成与分辨率边界

### 第二章 经典DOA估计算法（已发布）

- 掌握 MUSIC、ESPRIT、Capon/MVDR 等核心方法
- 会做模型阶数选择、相干源处理与性能评估

### 第三章 基于深度学习的DOA估计（已发布）

- 理解任务建模、数据集构建与主流网络范式
- 能对经典方法与深度学习方法进行对比选型

### 第四章 论文复现和工程实践（计划中）

- 聚焦顶刊/顶会论文复现（按黑客松形式持续发布揭榜清单，由同学们完成复现和PR提交）
- 毫米波实测与 Sim2Real 等问题


## 教程特点

- 阶梯式学习路径：从“为什么”到“怎么做”，再到“如何验证”
- 理论实验联动：每章围绕问题组织公式、代码和结果解释
- 可复现导向：强调可运行的实验路径与结果对照
- 教学叙事导向：尽量减少资料堆砌，突出问题驱动学习
